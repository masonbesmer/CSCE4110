Windows only run: .\DAC4110.exe <filename>

Alternative for other OS:
Install Go. https://golang.org/doc/install
go run . <filename>

<filename> is relative path to a file containing numbers separated by commas and commas only. NO SPACES.
Each new line is treated as a new array to search. The 1st value on each line will be treated as the
number to search for in the rest of the line (array)
